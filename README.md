# 👷 공수 플러스 (Gongsu Plus)

현장 근로자 및 일용직 근로자를 위한 가장 편리하고 스마트한 공수 및 급여 계산기입니다.

## ✨ 주요 기능

- **직관적인 달력 UI**: 날짜별 공수(0.5, 1.0, 1.5, 2.0, 휴무)를 터치 한 번으로 기록
- **직접 입력 기능**: 특수 공수(예: 2.5, 3.0 등)를 숫자로 직접 입력 가능
- **자동 급여 계산**: 설정한 단가와 세율(3.3% 등)을 바탕으로 월별 예상 수령액 자동 산출
- **실시간 데이터 동기화**: Google 로그인을 통한 Firebase 연동으로 기기 변경 시에도 데이터 보존
- **반응형 디자인**: 모바일과 PC 어디서나 최적화된 화면 제공

## 🚀 시작하기

1. [라이브 데모 사이트](https://ais-pre-qvjm5hwvn2ionpxkptbail-319674759917.asia-northeast1.run.app)에 접속합니다.
2. 구글 계정으로 로그인합니다.
3. 설정 메뉴에서 본인의 일당(단가)을 입력합니다.
4. 달력에서 날짜를 선택해 공수를 기록하세요!

## 🛠 기술 스택

- **Frontend**: React, TypeScript, Tailwind CSS, Motion
- **Backend/Database**: Firebase (Authentication, Firestore)
- **Deployment**: Google Cloud Run

## 📄 라이선스

이 프로젝트는 Apache-2.0 라이선스를 따릅니다.
